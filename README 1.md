## Quick Start (Local)
1. Clone the repo  
   `git clone https://github.com/WistantKode/<repo>.git`
2. Install dependencies  
   `npm install`
3. Run development server  
   `npm run dev`
4. Build for production  
   `npm run build`

(Replace `<repo>` with the actual repository name.)
